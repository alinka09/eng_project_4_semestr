Центр карьеры студентов Московского Политеха

Центр карьеры обучающихся создан с целью повышения конкурентоспособности студентов и выпускников Московского Политеха на рынке труда. Уделяя особое внимание построению диалога "Предприятие - Университет", специалисты подразделения создают условия, позволяющие обучающимся и выпускникам Московского Политеха узнать о карьерных возможностях, повысить уровень своих профессиональных навыков и надпредметных компетенций и определиться в выборе сферы своего профессионального развития.

Технологии

React.JS

JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов. React может использоваться для разработки одностраничных и мобильных приложений.

Bootstrap

свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения.

RESTfull API

архитектурный стиль взаимодействия компонентов распределённого приложения в сети. REST представляет собой согласованный набор ограничений, учитываемых при проектировании распределённой гипермедиа-системы. В определённых случаях это приводит к повышению производительности и упрощению архитектуры.

HTML and CSS

HTML — стандартизированный язык разметки документов во Всемирной паутине. CSS — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.

SQLlite

компактная встраиваемая СУБД. Исходный код библиотеки передан в общественное достояние. В 2005 году проект получил награду Google-O’Reilly Open Source Awards.

Django

свободный фреймворк для веб-приложений на языке Python, использующий шаблон проектирования MVC. Проект поддерживается организацией Django Software Foundation. Сайт на Django строится из одного или нескольких приложений, которые рекомендуется делать отчуждаемыми и подключаемыми.

Цель проекта

Целью данного курсового проекта является разработка веб-ресурса Центра карьеры студентов Московского Политеха. Проектирование и создание реляционной базы данных для веб-сервиса или приложения по предметной области, которая предложена нам по варианту. Создать и настроить административную панель для управления базой данных с помощью фреймворка Django. Изучить формат Json и реализовать RESTfull API. А также создание личного кабинета для трех пользователей. Реализация CRUD.

Он будет включать в себя базу данных, административную панель на Django с тремя ролями: пользователь-студент (users), пользователь-компания (personal), администратор сайта (admins). Также будет создан личный кабинет для этих ролей.
